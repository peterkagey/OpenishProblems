  Consider Ron Graham's sequence for $\operatorname{lcm}$, that is, look at sequences such that \[
    n = a_1 < a_2 < \hdots < a_T = k \text{ and } \operatorname{lcm}(a_1,\hdots,a_T) \text{ is square.}
  \]
\begin{question}
  Let $A300516(n)$ be the least k (as a function of n) such that such a sequence
  exists?
\end{question}
\begin{figure}[!h]
  \centering
  \begin{alignat*}{2}
  a(1) &= 1  &&\text{ via } (1)\\
  a(2) &= 4  &&\text{ via } (2, 4)\\

  }
